# Simple Calculator

## Description
A simple console-based calculator in C that performs addition, subtraction, multiplication, and division using functions.

## Features
- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers with error handling
- Loops until the user chooses to exit

## How to Run
1. Open terminal/command prompt
2. Navigate to the Simple-Calculator folder
3. Compile using: gcc calculator.c -o calculator
4. Run using: ./calculator

## Technologies Used
- C programming
- GCC compiler
